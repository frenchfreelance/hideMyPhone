hideMyPhone jQuery plugin

A jQuery plugin that helps you to hide your phone on your page and prevent crawlers to get it!

How to use ?

It's super simple.
you can use data attributes to make sure crawlers won't parse it easily, just like this
<a class="phone" href="" data-i="+33" data-r="783688537" data-dp1="07" data-dp2="83 68 85 37"

See how data attributes are used :

  • data-i : indicative
  • data-r : phone without indicative
  • data-dp1 : first part of the text of the phone
  • data-dp2 : second part of the text of the phone

One last thing, if you don't put anything between the <a><a>, your phone is printed by default.

Javascript / jQuery

Include in the bottom of your page jQuery and the plugin, for instance : 

<script src="js/jquery.min.js"></script>
<script src="js/jquery.hideMyPhone.min.js"></script>
Then, the plugin itself : <script src="js/jquery.hideMyPhone.min.js"></script>

Finally, either put just after this or in another javascript file include and call the function on a jQuery selector :
$(document).ready(function(){ $('a.mail').hideMyPhone(); });
and voilà!
